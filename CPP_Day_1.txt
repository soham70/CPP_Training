Q-Basic Structure of CPP
=> #include <iostream>
using  namespace std;

int main()
{
    cout << "Hello World!";
    return 0;

# include <iostream> :It is a preprocessor directive. It tells the preprocessor to include the contents of the iostream header file in the program before compilation. 
This file is required for input-output statements.

using namespace std: namespace contains all the identifiers of the C++ Standard Library. Using namespace std allows you to avoid prefixing 
standard library identifiers with std::

int main(): program startup or the entry point of a  program
